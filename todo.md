# مهام تطبيق حاسبة العملة السورية

- [x] واجهة المستخدم الرئيسية (شاشة الحاسبة)
- [x] حقول الإدخال (المبلغ المراد دفعه والمبلغ المدفوع)
- [x] المنطق الحسابي (حساب الفرق والترجيع)
- [x] عرض النتائج (الفرق بالعملتين)
- [x] اقتراحات الترجيع الذكية (مع صور العملات)
- [x] اقتراحات الدفع البديلة
- [ ] عرض صور العملات الجديدة
- [ ] عرض صور العملات القديمة
- [x] دعم الشكلين المختلفين للـ 500 القديمة

## الميزات الإضافية

- [ ] تنبيهات للحالات الخاصة (فرق أقل من 10 ليرات جديدة)
- [x] تنسيق الأرقام بفواصل آلاف
- [x] اختبار شامل للحسابات
- [ ] تحسين الواجهة والتصميم

## الجودة والاختبار

- [x] اختبار جميع الحالات الحسابية
- [ ] اختبار عرض صور العملات
- [ ] اختبار الاقتراحات الذكية
- [x] التأكد من عمل التطبيق دون إنترنت
- [ ] اختبار على أجهزة مختلفة (iOS و Android و Web)


## التعديلات الجديدة

- [x] تحديث الألوان إلى الزيتي والذهبي (الهوية البصرية السورية)
- [x] إضافة خيار اختيار العملة (جديدة أو قديمة) للدفع
- [x] عرض صور العملات الفعلية في الاقتراحات
- [x] تصحيح منطق الفئات - عدم استخدام 5 ليرات في الجديدة
- [x] إضافة انيميشنات انتقالية عند ظهور النتائج
- [x] حفظ سجل العمليات الحسابية محلياً
- [x] تحسين عرض العملات في الوضع المظلم
- [x] اقتراحات ذكية للفرق الصغير

## الإصلاحات المطلوبة

- [x] جعل لوحات المفاتيح أرقام فقط
- [x] إصلاح الخيار الثالث لعرض مزيج من العملة القديمة فقط
- [x] عرض صور العملات بشكل صحيح
- [x] تحسين الفسيفساء الذهبية لتكون أكثر وضوحاً
- [x] إضافة معاين للعملات لعرضها بحجم أكبر
- [x] زر مشاركة عبر واتساب ونسخ للحافظة
